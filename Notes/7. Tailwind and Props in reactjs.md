## ğŸ’¡ What are Props in React?

### ğŸ§¾ Definition:

**Props** (short for **properties**) are used to pass data **from a parent component to a child component**.

---

### ğŸ”„ Why use props?

| Benefit                 | Description                                     |
| ----------------------- | ----------------------------------------------- |
| âœ… Reusability           | Customize one component in multiple ways        |
| ğŸ“Š Dynamic UI           | Component output changes based on passed props  |
| âš™ï¸  Separation of Logic | Logic stays in parent; UI rendering is in child |

---

### ğŸ“¥ How to Use Props (with Example)

#### 1. **Passing Props from Parent to Child**

```jsx

  return (
    <>
      <Card username="Owl-1" btnText="click me" />
```

```jsx

  return (
    <div>
      <h2>{username}</h2>
      <button>{btnText}</button>
    </div>
  );
}
```

### What are Props in React?

**Props** (short for **properties**) are a way of passing **data from one component to another** in React. They allow you to make components **reusable** and **dynamic** by customizing their behavior or display based on the data you provide.

---

### ğŸ”§ How Props Work:

* Props are passed **from a parent component to a child component**.
* They are **read-only** inside the child component (i.e., you can't modify them directly in the child).
* Props are passed like **HTML attributes**.

---

### âœ… Example:

```jsx
// ParentComponent.jsx
import ChildComponent from './ChildComponent';

function ParentComponent() {
  return <ChildComponent name="Prabhan" age={21} />;
}
```

```jsx
// ChildComponent.jsx
function ChildComponent(props) {
  return (
    <div>
      <h1>Hello, my name is {props.name}</h1>
      <p>I am {props.age} years old.</p>
    </div>
  );
}
```

---

### ğŸ§  Why Use Props?

1. âœ… **Reusability** â€“ You can use the same component in different places with different data.
2. ğŸ” **Dynamic UI** â€“ Components change output based on passed data.
3. ğŸ“¦ **Separation of Concerns** â€“ Parent holds the logic, children display the data.

---

### ğŸ“Œ Destructuring Props (Cleaner Code):

```jsx

  return (
    <div>
      <h1>Hello, my name is {name}</h1>
      <p>I am {age} years old.</p>
    </div>
  );
}
```

---

### ğŸ§ª Interview Tip:

* **Q:** Are props mutable or immutable in React?
* **A:** Props are **immutable** â€“ you can't change them inside the component receiving them.
